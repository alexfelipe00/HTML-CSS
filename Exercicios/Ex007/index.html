Nessa frase temos um termo em negrito usando a tag <strong> (semântica).

Itálico

Nessa frase temos um termo em itálico usando a tag <i> (não semântica).

Nessa frase temos um termo em itálico usando a tag <em> (semântica).

Texto marcado

Nessa frase temos um texto marcado usando a tag <mark> (semântica).

Texto grande e pequeno

Texto grande com a tag <big> e um texto pequeno nesse parágrafo utilizando a tag <small>

Texto deletado

Marcar um texto como deletado para indicar que ele deve ser lido, mas não considerado utilizando a tag <del>.

Texto inserido

Marcar um texto como inserido para dar ênfase e indicar que ele foi adicionado depois usando a tag <ins> (semântica).

Existe também o sublinhado com a tag <u> (não semântica).

Texto sobrescrito

Para inserir coisas do tipo x2+4x-2utilizando a tag <sup>.

Texto subscrito

Para inserir coisas do tipo H20 utilizando a tag <sub>.
